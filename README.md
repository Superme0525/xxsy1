##信息素养实践课程  
###什么是信息素养实践？  
**《信息素养实践》**课程作为软件工程专业的入门课程，属于*专业必修课*。  
###开设此课程的意义  
*信息素养实践课程*的目的是使学生掌握计算机的基本使用以及常见应用软件的使用。  
###计分标准  
该课程采用的过程化考核方案为：雪梨（平台）作业占55%，平时成绩占15%，期末考核占30%。具体细化如下：  
|作业|内容|所占分数|
|:-:|:-|-:|
|第一次作业|常见DOS命令的使用|10分|
|第二次作业|markdown文档|10分|
|第三次作业|Github的使用及命令|10分|
|第四次作业|思维导图|10分|
|第五次作业|word应用技巧|8分|
|第六次作业|excel应用技巧|7分|
